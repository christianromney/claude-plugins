# claude-plugins

Personal Claude Code plugin marketplace for [Christian Romney](https://github.com/christianromney).

This repository serves as a private plugin marketplace. Each top-level directory is a plugin containing one or more Claude Code skills. Install the marketplace with:

```bash
claude plugins add https://github.com/christianromney/claude-plugins
```

## Plugins

| Plugin | Skill | Description |
|---|---|---|
| [usage-insights](usage-insights/) | `/usage-insights` | Generate time-aware, config-aware HTML usage insight reports from Claude Code session data |
